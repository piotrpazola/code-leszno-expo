## Getting Started

1. Remove existing version control

   ```sh
   rm -R .git
   ```

2. Install dependencies

   ```sh
   yarn install
   ```

3. Start the webpack server. The server will run at [`localhost:3000`](http://localhost:3000).

   ```sh
   yarn start
   ```
